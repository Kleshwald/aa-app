# Логотипы страховых компаний

Сюда положить квадратные логотипы СК (PNG, лучше на прозрачном/белом фоне, ~512×512).
`InsurerLogoComponent` ищет файл `/{id}.png`; если файла нет — показывает монограмму-фолбэк.

Имена файлов = id страховой компании. Сохрани присланные логотипы так:

| Файл | СК | Как выглядит присланный лого |
|------|----|------------------------------|
| `renessans.png` | Ренессанс страхование | фиолетовый квадрат + салатовый треугольник |
| `soglasie.png` | Согласие | оранжевые переплетённые кольца |
| `ugoria.png` | Югория | синяя «U» из трёх линий |
| `zetta.png` | Зетта | бирюзовый ромб с белой «Z» |
| `sogaz.png` | СОГАЗ | тёмно-синий круг с белой каплей |
| `rosgosstrah.png` | Росгосстрах | бордовый квадрат «РГС» |
| `euroins.png` | Евроинс | синий круг со звёздами и полосами |
| `energogarant.png` | Энергогарант | оранжевый квадрат с белой «Э» |

Для текущих флоу нужны минимум: `renessans`, `soglasie`, `ugoria` (Здоровье) и
все 8 для ОСАГО. Без файлов всё работает — показывается монограмма.
