# Agent Academy — Прототип

Кликабельный прототип нового фронтенда платформы Agent Academy.

## С чего начать

### Шаг 1. Подготовь инструменты (один раз)

1. Установи **Node.js LTS** (20+) — https://nodejs.org
2. Установи **Angular CLI**: `npm install -g @angular/cli`
3. Установи **Cursor**: https://cursor.com (нужна Pro/Business подписка)
4. В настройках Cursor выбери модель **Claude Opus 4.6** (или последнюю Opus)
5. В настройках Cursor включи **Agent Mode** и **Auto-apply**

### Шаг 2. Открой эту папку в Cursor

`File → Open Folder → выбрать папку проекта`

### Шаг 3. Запусти первый промпт в Cursor

Открой Composer (`Cmd+I` / `Ctrl+I`), убедись что выбран **Agent Mode**.

Отправь промпт из файла `FIRST_PROMPT.md`.

## Ключевые решения (зафиксированы)

- **Фронтенд:** Angular 18 + TypeScript strict
- **UI-библиотека:** Taiga UI 5
- **Стилизация:** Tailwind CSS (layout) + Taiga (компоненты)
- **Архитектура:** Angular напрямую к HTTP-сервисам 1С (без middle-слоя в V1)
- **Mock-данные:** через HttpInterceptor (Faker.js)
- **Авторизация:** JWT
- **PWA:** да, installable, basic offline
- **Десктоп-first:** основной фокус (минимум 1366×768)
- **Мобилка:** PWA для конкретных сценариев (пролонгации)

## Целевая аудитория (важно для дизайна!)

- **Женщины 35-60 лет** (68%)
- **Регионы РФ** — Сибирь, Дальний Восток, Урал (97% НЕ в крупных городах)
- **Старые компьютеры**, Windows, иногда плохой интернет
- **Стиль интерфейса:** профессиональный российский B2B (Госуслуги, Сбербанк Онлайн)
- **НЕ:** Apple-style, Linear, Vercel — слишком модно

См. `USER_PROFILE.md` для деталей.
